# Metro SPB

Программа для расчета маршрутов метро Санкт-Петербурга.  
Использует алгоритм Дейкстры для поиска кратчайшего пути между станциями с учетом времени движения и пересадок.

## Функционал

- Инициализация метро из JSON-файла (`metro.json`)
- Построение графа метро в виде **матрицы смежности**
- Алгоритм Дейкстры для поиска **кратчайшего пути по времени**
- Учет пересадок между линиями
- Вывод маршрута с указанием:
    - Линий метро
    - Названий станций
    - Общего времени поездки

---

## Требования

- Java 17+ (или совместимая версия)
- Компилятор `javac`
- Maven или Gradle не обязательны (проект можно компилировать вручную)
- `metro.json` должен находиться в `src/main/resources/`

---

## Как запустить


1. Компиляция

javac -d out src/main/java/org/example/metro/**/*.java

2. Копирование ресурсов

cp src/main/resources/metro.json out/org/example/metro/

3. Запуск программы

java -cp out org.example.metro.Main
